## 5.Printing the size of the file
```
#include <stdio.h>
#include <unistd.h>
#include <fcntl.h>
#include <sys/types.h>
#include <sys/stat.h>
int main(){
 int fd;
 struct stat buf;
 fd=open("file.txt",O_RDWR);
 fstat(fd,&buf);
 printf("%ld",buf.st_size);
 close(fd);
}
```
## 6.Write a C program to create a new text file and write "Hello, World!" to it?
